# Contributing to MendelCell

Thank you for your interest in MendelCell.

MendelCell is a research software tool for exploratory candidate gene prioritization using tissue-specific and cell-type-specific expression data.

## Reporting Bugs

If you find a bug, please open a GitHub issue and include:

* A brief description of the problem
* Steps needed to reproduce the issue
* The input file format used, if relevant
* The tissue name and threshold used, if relevant
* Any error message shown by the app

Please do not upload protected health information, patient-identifiable data, credentials, or other sensitive data.

## Requesting Features

Feature requests are welcome through GitHub issues. Please describe:

* The feature you would like added
* Why it would be useful
* Example input/output, if applicable

## Contributing Code

Contributions can be made by opening a pull request.

Before submitting a pull request, please:

1. Make sure the app runs locally.
2. Run the test suite.
3. Add or update tests if your change affects analysis logic.
4. Keep code and documentation clear and readable.

To run tests locally:

```bash
python -m pytest
```

## Research Use Only

MendelCell is intended for research and exploratory analysis only. It is not a diagnostic tool and should not be used to make clinical decisions.
